# ランダム名前選び

カンマ区切りで入力された名前からランダムに1つを選択するWebアプリケーション。

## 機能

- カンマ区切りの名前を入力
- ボタンを押すとランダムに1つの名前を選択
- クリアボタンで結果をリセット

## 技術スタック

- **フレームワーク**: React 19.2.0
- **言語**: TypeScript 5.9.3
- **ビルドツール**: Vite 7.3.1
- **テスト**: Vitest + React Testing Library

## セットアップ

```bash
# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev

# ビルド
npm run build

# テスト
npm run test

# テストUI（Vitest UI）
npm run test:ui
```

## 使い方

1. テキストエリアに「田中, 山田, 佐藤」のようにカンマ区切りで名前を入力
2. 「🎲 選ぶ」ボタンをクリック
3. ランダムに選ばれた名前が表示されます
4. 「クリア」ボタンで結果をリセット
